what tim does

reads & edits files

surgical string edits, no full-file rewrites. glob, grep, read, write, list.

runs shell commands

bash execution with timeouts. ctrl-c safe. stdout streamed back.

spawns sub-agents

sequential or parallel swarms with a shared workspace. one tim becomes many.

browses the web

page fetch with content extraction, plus tavily-powered search.

sends email

send, receive, and schedule mail via agentmail. tim will email you.

remembers

persistent memory in ~/.tim. agents, workflows, sessions, triggers.

schedules itself

cron-style triggers and workflows. tim shows up on time.

speaks MCP

plug in any model context protocol server. tim plays nice.

iOS app

chat with tim on the go. download from the app store.

install

# clone, link, set your key — that's it
git clone https://github.com/tmoreton/heytim.git && cd heytim
npm link

# set your openrouter api key
tim /env set OPENROUTER_API_KEY=sk-...

# start talking
tim

philosophy

small codebase, full reach. ~5,000 lines and zero deps — yet tim spawns sub-agent swarms, browses the web, sends email, speaks MCP, and wakes himself up on a cron. no node_modules black hole. read it on a plane. fork it on a napkin. when the agent future gets weird, you'll be glad tim fits in your head.
